# INFOGRAPHIC_PROMPT

## 1. 全体構成（レイアウト）
- アスペクト比: 16:9（横長）
- 上部20%: 大見出しを配置
- 中央部40%: 左に「原因」、中央に「状況」、右に「結果」を3分割で配置
- 下部30%: 「3つのポイント」を箇条書きで配置
- 右下10%: 「投資家Fキャラ」と「吹き出し」を配置
- 全体: 余白を多く取り、スマホでも読みやすいように文字は大きめ

## 2. デザインとスタイル
- 全体テーマ: 「かわいい＋情報整理」
- 色使い: パステルカラー基調（メイン: 水色、アクセント: ピンク、クリーム色）
- フォント: 手書き風の丸ゴシック体
- 装飾: 雲、星、キラキラなどのあしらいを控えめに配置。各ボックスは角丸の四角形で囲む。

## 3. 各要素の詳細
- **大見出し**: 「{HEADLINE}」というテキストを一番大きく表示。
- **原因ボックス**: 
    - 小見出し: 「原因は？🤔」
    - テキスト: 「{CAUSE}」
- **状況ボックス**:
    - 小見出し: 「今どうなってるの？🗺️」
    - テキスト: 「{SITUATION}」
- **結果ボックス**:
    - 小見出し: 「どうなるの？→」
    - テキスト: 「{RESULT}」
- **3つのポイント**: 
    - 全体タイトル: 「💡 3つのポイント」
    - POINT 1: 「{POINT1}」
    - POINT 2: 「{POINT2}」
    - POINT 3: 「{POINT3}」
- **投資家Fキャラ**: 
    - キャラクター: 銀髪の女の子、猫耳フード付きの青いパーカーを着ている。デフォルメされた可愛いスタイル。
    - ポーズ: 指差しポーズでポイントを指し示している。
    - 吹き出し内のセリフ: 「{F_COMMENT}」

## 4. 禁止事項
- 複雑なグラフや表の使用
- 暗い色、原色の使用
- 攻撃的、断定的な表現
- 投資助言、売買推奨、銘柄推奨
