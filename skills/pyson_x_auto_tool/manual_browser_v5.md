_# Project Prometheus ブラウザ自動いいね v5.0 - かんたんマニュアル

こんにちは！これは、あなたのX（旧Twitter）アカウントの「自動いいね」をお手伝いするプログラム「Prometheus（プロメテウス）」の新しいマニュアルです。v5.0は、これまでのAPI方式から**ブラウザ自動化方式**に変わりました。

## 1. 何が変わったの？

一言でいうと、**APIを使わず、本物のブラウザ（Chrome）を操作して「いいね」するようになりました。**

これにより、Xの厳しいAPI制限（15分で5回）を気にする必要がなくなり、より多くの「いいね」を、より人間らしい自然なペースで行えるようになります。

### v5.0のすごいところ

- **API制限なし**: 15分5回の制限を受けません。
- **人間らしい動作**: 5秒〜15秒のランダムな間隔で「いいね」します。
- **安全なログイン**: 初回に手動でログインすれば、その情報が安全に保存され、次回からは自動でログインします。

## 2. 使い方

### ステップ1: 初回セットアップ（最初の1回だけ）

まず、プログラムにXのアカウントを覚えさせる必要があります。

1.  **ターミナルを開く**
2.  以下のコマンドを実行します:
    ```bash
    cd ~/prometheus
    python3 auto_like_browser.py --setup
    ```
3.  自動でブラウザが立ち上がるので、**Xにログイン**してください。
4.  ログインが完了したら、ターミナルに戻って`Enter`キーを押します。

これでログイン情報が保存され、次回からこの作業は不要です。

### ステップ2: 自動いいねを開始

デスクトップに「**自動いいね_ブラウザ版.command**」というファイルがあります。これをダブルクリックするだけで、自動いいねがスタートします。

<img src="https://i.imgur.com/J0tF7aC.png" alt="ショートカットアイコン" width="200"/>

実行すると、黒い画面（ターミナル）が出てきて、プログラムが動き始めます。この画面は閉じないでください。

### やめたいときは？

黒い画面を選択した状態で、キーボードの `control` キーを押しながら `C` のキーを押してください。プログラムが安全に停止します。

## 3. 注意してほしいこと

- このプログラムは、バックグラウンドでブラウザを動かします。実行中は、DockにChromeのアイコンが表示されることがありますが、正常な動作です。
- API制限はありませんが、短時間にあまりにも多くの「いいね」をしすぎると、Xからアカウントが怪しまれる可能性があります。プログラムには適切な間隔が設定されていますが、長時間の連続使用は避けた方が安全です。

何か分からないことがあったら、いつでも聞いてくださいね！_
