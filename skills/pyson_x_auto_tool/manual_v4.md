# Project Prometheus 自動いいねシステム v4.0 - かんたんマニュアル

こんにちは！これは、あなたのX（旧Twitter）アカウントの「自動いいね」をお手伝いするプログラム「Prometheus（プロメテウス）」のマニュアルです。中学2年生にも分かるように、やさしく解説しますね。

## 1. このプログラムで何ができるの？

一言でいうと、**あなたの代わりに、Xで自動的に「いいね」をしてくれるプログラム**です。

あなたが興味のあるキーワード（FX、投資、経済など）が含まれる投稿を自動で見つけて、「いいね」を押してくれます。これにより、あなたのアカウントが他の人に見てもらえる機会が増え、フォロワーが増えるきっかけになるかもしれません。

## 2. なぜ新しいバージョンになったの？

前のバージョンでは、Xのルールが厳しくなったことで、うまく「いいね」ができない問題が起きていました。

Xには、「API」というプログラムからXを操作するための仕組みがあるのですが、そのルールが変更され、**「15分間に5回までしか、いいねできませんよ」**という厳しい制限ができてしまったのです。

この新しいルールに対応するために、プログラムを根本から作り直したのが、このv4.0です。

### v4.0のすごいところ

- **ルールをしっかり守る**: 15分間に5回以上いいねしないように、自分でしっかり数えています。
- **賢く待つ**: 1回いいねしたら、次のいいねまで約3分待ちます。（15分 ÷ 5回 = 3分）
- **エラーに強い**: もしXから「ちょっと待って！」と言われても、言われた時間だけしっかり待ってから再開します。

## 3. どうやって使うの？

デスクトップに「**自動いいね開始.command**」というファイルがあります。これをダブルクリックするだけで、自動いいねがスタートします。

<img src="https://i.imgur.com/J0tF7aC.png" alt="ショートカットアイコン" width="200"/>

実行すると、黒い画面（ターミナル）が出てきて、プログラムが動き始めます。この画面は閉じないで、そのままにしておいてください。閉じるとプログラムも止まってしまいます。

### 実行中の画面の見方

- `[INFO]`：プログラムが今何をしているかのお知らせです。
- `[SUCCESS]`: 「いいね」に成功した合図です！
- `[WARNING]`: 何か注意点があったときに出ます。「レート制限」という表示が出たら、Xのルールに従って休憩している証拠です。
- `[ERROR]`: 何か問題が起きたときに出ます。もし何度も出るようなら、サポートに連絡してください。

### やめたいときは？

黒い画面を選択した状態で、キーボードの `control` キーを押しながら `C` のキーを押してください。プログラムが安全に停止します。

## 4. 注意してほしいこと

- このプログラムは、あなたがXにログインしている「おすすめタイムライン」の投稿にいいねをします。
- 1日に「いいね」できる数には上限があります。プログラムが自動で管理しますが、手動での「いいね」をしすぎると、プログラムが使える分が減ってしまうので気をつけてください。

何か分からないことがあったら、いつでも聞いてくださいね！
